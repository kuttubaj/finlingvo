# FinLingvo

FinLingvo is a demo MVP concept prototype for learning business and financial literacy through a Business City simulation.

The core idea: a user develops a business city by entering buildings, completing missions, and growing capital, reputation, and level. Each building represents a learning module, while missions are short practical business cases.

## Why It Is Not Duolingo-Style

FinLingvo avoids the familiar language-learning path pattern: no vertical lesson road, no round lesson nodes, no hearts or lives, and no cartoon green visual language. The experience is designed as a modern dark business simulator with city buildings, module cards, mission cases, capital, and reputation.

## Screens

- HomePage: Business City overview, user stats, building cards, and today's mission.
- BankPage: Bank building module with progress and mission cards.
- MissionPage: Business case with expenses, answer choices, and feedback.
- ResultPage: Success state with formula explanation and business rewards.

## Run Locally

```bash
npm install
npm run dev
```

## Notes

This is a frontend-only demo MVP built with React, Vite, JavaScript, and regular CSS. It has no backend, no authentication, no database, and no production data storage.
